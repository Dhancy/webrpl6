CSS (Cascading Style Sheet)

HTML adalah kerangka dari layout sebuah website. Bisa diibaratkan sebagai tubuh manusia yang tidak menggunakan apa-apa. Tanpa Baju, tanpa aksesoris, tanpa Make up. Sedangkan fungsi dari CSS adalah sebagai baju, make up, dan aksesoris

Fungsi dari css mempercantik tampilan web, sedangkan HTML sebagai struktur utamanya.
